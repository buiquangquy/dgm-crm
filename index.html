🔌 Kết nối Google Sheets
Dán URL Apps Script Web App để kết nối dữ liệu persistent 2 chiều.
1
Mở Google Sheet → Extensions → Apps Script → paste code từ file apps-script-backend.js
2
Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone → Deploy
3
Copy Web app URL (dạng https://script.google.com/macros/s/...) và dán vào đây:
